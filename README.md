# barhandler-manager

Local hardware bridge for Barhandler, FitStudio, and other products.

Runs on `localhost:9999` and accepts print/terminal commands from any web app via HTTP API.

## Supported hardware
- Thermal receipt printers (ESC/POS)
- Label printers (ZPL / TSPL)
- POS terminals *(planned)*

## Quick start
```bash
pip install -r requirements.txt
python main.py
```
